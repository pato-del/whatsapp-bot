# WhatsApp Bot Ultra 🤖

A powerful WhatsApp bot with downloads, EPL, stickers, and group tools.

## Features

* Song downloader
* Video downloader
* TikTok downloader
* EPL standings
* Group moderation
* Stickers
* Fun commands

## Installation

```bash
git clone https://github.com/YOURNAME/whatsapp-bot-ultra.git
cd whatsapp-bot-ultra
npm install
node index.js
```
